# Анализ продаж супермаркета (SQL + Python)

## О проекте
Учебный проект, в котором проведен **полный анализ данных** о продажах вымышленного супермаркета. Цель проекта — продемонстрировать **практические навыки работы с данными**, начиная от их изучения и заканчивая формулированием **бизнес-выводов**.

## Что сделано
1. **Обработка и изучение данных** с использованием **Pandas**
2. **Расчет ключевых метрик**: общая выручка, прибыль, средний чек
3. **Анализ по категориям** и выявление **самых прибыльных товаров**
4. **Исследование временных рядов** — анализ динамики продаж по месяцам
5. **Бизнес-анализ**: правило **20/80 для клиентов**, влияние скидок на прибыль
6. **Построение дашборда** с 4 графиками с использованием **Matplotlib/Seaborn**
7. **Написание SQL-запросов** для аналогичных расчетов

## Технологический стек
- **Python**: Pandas, NumPy, Matplotlib, Seaborn
- **SQL**: оконные функции, агрегация, GROUP BY
- **Анализ данных**: временные ряды, сегментация, статистика

## Ключевые выводы из анализа
- **Сезонность продаж**: пик в ноябре-декабре (+40% к среднему месяцу)
- **Влияние скидок**: скидки более 20% часто приводят к убыткам
- **Категория "Technology"**: дает 45% прибыли при 20% продаж
- **Лояльность клиентов**: 15% клиентов совершают повторные покупки

## Навыки, продемонстрированные в проекте
- **Обработка и очистка данных**
- **Аналитическое мышление** и работа с метриками
- **Визуализация данных**
- **SQL** на уровне написания аналитических запросов
- **Формулирование бизнес-выводов** на основе данных
